# 🧪 FizzBuzz Kata

## 📌 Descripción

Este proyecto implementa el clásico ejercicio **FizzBuzz**

---

## 🎯 Objetivos del ejercicio

- Practicar **Test-Driven Development (TDD)**.
- Practicar tests unitarios
- Aplicar buenas prácticas en la escritura de tests.

---

## Comportamiento

| Entrada        | Salida     |
|----------------|------------|
| 1              | "1"        |
| %3==0          | "Fizz"     |
| %5==0          | "Buzz"     |
| %3==0 && %5==0 | "FizzBuzz" |
